# Техническое задание для разработки дашборда

## 1. Общая информация

| Название | Значение |
|---|---|
| **Название отчёта** | Данные о продажах медицинских пробирок |
| **Бизнес-цель отчёта** | Получение оперативной информации о продажах для принятия решений |
| **Подразделение заказчика** | Отдел продаж |
| **Заказчик** | Директор отдела продаж |
| **Целевая аудитория дашборда** | Директор отдела продаж, менеджеры по работе с поставщиками и менеджеры по продажам |

## 2. Показатели / метрики

| Название показателя | Логика расчёта | Источник данных | Комментарий |
|---|---|---|---|
| Сумма продаж | Сумма по полю `total_sale_amount` | База данных | |
| Количество проданных товаров | Сумма по полю `quantity` | База данных | |
| Средний чек | Сумма продаж / количество проданных товаров | База данных | |
| Количество регионов продаж | Уникальные значения по полю `region_id` | База данных | |
| Координаты регионов продаж | Новое поле, рассчитанное по формуле: `GEOPOINT([longitude], [latitude])` | База данных | |
| Продажи за отчетный период | Новое поле, рассчитанное по формуле: `SUM(IF(DATE([Дата продажи]) BETWEEN [base_date_from] AND [base_date_to], [total_sale_amount], NULL))` | База данных | |
| Продажи за период сравнения | Новое поле, рассчитанное по формуле: `SUM(IF((DATE([Дата продажи]) BETWEEN [compare_date_from] AND [compare_date_to]) AND [total_sale_amount] IS NOT NULL, [total_sale_amount], NULL))` | База данных | |
| Динамика между отчетным периодом и периодом сравнения, в руб. | Новое поле, рассчитанное по формуле: `[Продажи за отчетный период] - [Продажи за период сравнения]` | База данных | |
| Динамика между отчетным периодом и периодом сравнения, в % | Новое поле, рассчитанное по формуле: `CONCAT(IF([Продажи за отчетный период] - [Продажи за период сравнения] > 0, "✔️ ", "❌ "), STR(ROUND(([Продажи за отчетный период] - [Продажи за период сравнения]) / [Продажи за период сравнения] * 100, 1)), "%")` | База данных | |

## 3. Фильтры

| Название фильтра | Значения фильтра | Комментарий |
|---|---|---|
| Дата продажи | Диапазон от 01.01.2023 до 31.12.2024 | На 1 листе |
| Регион | Все значения поля `region_name` | На 1 листе |
| Тип клиента | Все значения поля `customer_type` | На 1 листе |
| Канал продаж | Все значения поля `sales_channel` | На 1 листе |
| Регион | Все значения поля `region_name` | На 2 листе |
| Товар | Все значения поля `product_name` | На 2 листе |
| Поставщик | Все значения поля `supplier_name` | На 2 листе |
| Выбранный период | Начало / конец (параметр) | На 2 листе |
| Период сравнения | Начало / конец (параметр) | На 2 листе |

## 4. Визуализации

| График | Особенности (параметры) | Комментарий |
|---|---|---|
| Динамика суммы продаж (линейный график) | Тип: линейный график. Детализация: квартал/месяц/день | На 1 листе |
| Динамика среднего чека (линейный график) | Тип: линейный график. Детализация: квартал/месяц/день | На 1 листе |
| Индикаторы (4 шт.) | По 4 метрикам | На 1 листе |
| Корреляция категории товара и среднего чека | Тип: Пузырьковая диаграмма. Ось X: количество единиц; Ось Y: сумма продаж; Размер: средний чек; Цвет: по категориям | На 1 листе |
| Топ-10 товаров по количеству продаж | Тип: Линейчатая диаграмма. Цвет: по категориям | На 1 листе |
| Структура продаж по поставщикам и типу клиента | Древовидная диаграмма | На 1 листе |
| География продаж | Тепловая карта | На 1 листе |
| Топ по сумме продаж (линейчатая диаграмма) | Детализация: категория > товар, динамика в подписи | На 2 листе |
| Топ по количеству продаж (линейчатая диаграмма) | Детализация: категория > товар | На 2 листе |
| Детализация по поставщикам, типу клиента, каналу продаж (таблица) | Поля таблицы: Поставщик, Тип клиента, Канал продаж, Сумма за выбранный период, Сумма за период сравнения, Динамика (в абсолютных значениях), Динамика (в %) | На 2 листе |
